Welcome to ScurvyScript, ye scurvy dog!

ScurvyScript be a pirate-themed programming language, inspired by BASIC, that'll make ye feel like ye're sailing the seven seas of code.

Here be some basic commands to get you started:

  • hoist: Declare a variable, ye landlubber!
  • parley: Show yer message to the crew!
  • if, elif, else: Make decisions like a true captain!
  • while: Keep loopin' till ye find yer treasure!

Now set sail and start coding, ye scurvy dog!
