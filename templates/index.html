Welcome to ScurvyScript, ye scurvy dog!

ScurvyScript be a pirate-themed programming language that'll make ye feel like ye're sailing the seven seas of code.

Here be some basic commands to get ye started:

  • yeet: Declare a variable, ye landlubber!
  • print: Show yer message to the crew!
  • if, but_if, by_chance: Make decisions like a true captain!
  • with, or, nay: Combine yer thoughts, matey!

Now set sail and start coding, ye scurvy dog!
